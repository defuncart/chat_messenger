# chat_messenger

A simple chat messenger build using Flutter and Firebase.

## About

Presently targets Android, iOS and macOS.

## Getting Started

If you would like to build the app yourself, then you simply need:

- Flutter >= 1.22.0
- Dart >= 2.7
- Xcode >= 11.3
- Android SDK >= 29.0.4
